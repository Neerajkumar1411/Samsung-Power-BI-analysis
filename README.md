# Samsung-Power-BI-analysis
1 Samsung sales Analysis
An interactive Power BI dashboard visualizing Samsung’s mobile sales performance. It highlights key metrics — 769M total sales, 4K transactions, and 19K total quantity — with insights by brand, model, city, day, and payment method.
Features include:

purpose.
The purpose of creating this Samsung Mobile Sales Dashboard is to provide a clear, data‑driven view of business performance. It helps stakeholders quickly analyze sales trends, transaction volumes, and customer behavior across different dimensions — brand, model, city, and payment method.

1.Comparative analysis of Apple, Samsung, OnePlus, Vivo, and Xiaomi sales

2.Customer rating distribution and utility‑wise vehicle counts

3.Geographic sales mapping across Indian cities

4.Trend lines for daily quantity and sales patterns
the dashboard was build using the following
* SQL - for for finding insights to be shown in dashboard
* Power query - Data transformation  and cleaning layers for reshaping and preapring the data.
* Data modeling - establishing relationship among tables.
